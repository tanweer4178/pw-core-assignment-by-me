HTML, CSS, JavaScript
The foundational technologies for building the user interface and interactivity of web applications.
React, Angular
Front-end frameworks provide a structured way to build complex user interfaces and manage state.
Node.js
A runtime environment that allows JavaScript to run on the server side, enabling server-side scripting and building back-end APIs.
Express.js
A web application framework for Node.js that simplifies the creation of robust and scalable APIs.
MySQL, MongoDB
Stores and manages data for web applications, with relational databases and NoSQL databases.
RESTful APIs
A design pattern for creating web services that allow communication between the front end and back end of an application.
Version Control ( Git)
Tools like Git help developers track and manage changes to their codebase, enabling collaboration and version history.
Deployment and Hosting
Platforms and services for deploying web applications to make them accessible on the internet.
