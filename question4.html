List of html tag

  1. html the main element that cantain all other html element on the webpage
  2. head contain meta information about html document, such as the title and character
  3. title set the title of the html document, which apears in the browser title bar
  4. metaprovides metadata about the html document
  5. bodycontains the visible content of the html page, including text, image, and other enformation
  6. h1 to h6 represents a top level heading and down level heading is used for the main title of the page
  7. ol create an ordered number list of items
  8. li Represents a list item which an ordered or unordered list
  9. a defines a hyperlink that can link to another web page or resource.
  10. img insert an image in html document
